# 7z

## 介绍

7z也是一个解压文件比unzip更强大，当unzip不支持某种zip的压缩/加密方式，用7z试试，往往可以。

7z支持的算法更多

7z(7-Zip)支持

​	（1）普通ZIP

​	（2）AES加密ZIP

​	（3）ZIP64

​	（4）以及.7z/.rar/.iso等多种格式

## 参数

| 参数 | 作用             |
| ---- | ---------------- |
| -o   | 解压目录         |
| -y   | 自动确认         |
| -bd  | 安静模式         |
| -r   | 递归（压缩使用） |
| -p   | 密码“必须紧跟-p” |



## 解压

7z x file.zip

## 解压带密码

7z x file.zip -p'' -f

### 注意：-p（必须要紧跟密码）不要留空格
