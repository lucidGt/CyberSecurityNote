## readpst

### 介绍

readpst是listpst工具,用于将Outlook PST文件转换为mbox或其他格式(Linux命令行).

### 参数

| 参数   | 作用                           |
| ------ | ------------------------------ |
| -o dir | 输出目录                       |
| -r     | 递归模式                       |
| -S     | 单独文件模式(每个右键单独.eml) |
| -D     | 包含已删除项                   |
| -h     | 显示帮助                       |

readpst file.pst (默认输出mbox文件,按文件夹命名)

### 用法

递归为mbox

readpst -r -o output file.pst
